# Charity Project

## Purpose
The Charity Project aims to provide a platform for non-profit organizations to connect with donors and raise funds for a variety of charitable causes. It serves to streamline the process of donations, making it easier for people to contribute to causes they care about.

## Features
- **User Registration**: Users can sign up and create profiles.
- **Donation Management**: Organizations can create and manage donation campaigns.
- **Search Functionality**: Users can search for charities by cause or name.
- **Payment Integration**: Secure payment gateway for processing donations.
- **Reports and Analytics**: Organizations can access insights on fundraising efforts.

## Setup Instructions
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/mishalsalman/charity.git
   cd charity
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Environment Variables**: Create a `.env` file in the root directory and add your **API keys** and **database connection strings**.

4. **Run the Application**:
   ```bash
   npm start
   ```
   Open your browser and navigate to `http://localhost:3000`.
